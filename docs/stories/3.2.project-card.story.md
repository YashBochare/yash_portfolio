# Story 3.2: Design Project Card Component

## Status
Done

## Story
**As a** Developer,
**I want** A reusable project card component,
**so that** Projects display consistently with hover effects

## Acceptance Criteria
1. ProjectCard component created
2. Displays: thumbnail, title, description, tech tags
3. Hover effect: lift and shadow increase
4. onClick handler for modal
5. Responsive and dark mode support

## Tasks / Subtasks
- [ ] Task 1: Create Component (AC: 1)
  - [ ] Create `src/components/common/ProjectCard.jsx`
  - [ ] Props: `project` (object), `onClick` (func)
- [ ] Task 2: Implement UI (AC: 2, 5)
  - [ ] Image: Top section, fixed height (e.g., `h-48`), `object-cover`.
  - [ ] Content: Title (`font-bold`), Description (truncated if long), Tags (pills/badges).
  - [ ] Mode: Tailwind dark mode classes for bg and text.
- [ ] Task 3: Interactive States (AC: 3, 4)
  - [ ] Container: `cursor-pointer`, `transition-all`, `duration-300`.
  - [ ] Hover: `hover:-translate-y-2`, `hover:shadow-xl`.
  - [ ] Click: Call `onClick(project)`.

## Dev Notes
- **Tech Tags**: Use a simple flex wrap layout for tags. Styles should use `bg-primary/10 text-primary` for a modern look.
- **Truncation**: `line-clamp-2` or `line-clamp-3` for description to keep cards uniform height.

### Testing
- **Test File Location**: `src/components/common/__tests__/ProjectCard.test.jsx`
- **Standards**: Verify props rendering and click handler.
