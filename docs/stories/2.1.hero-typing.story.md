## Dev Notes
- **Framer Motion**: Standard library for React animations. [Source: docs/front-end-architecture.md#tech-stack]
- **Hook Logic**: Careful with `useEffect` dependencies to avoid infinite re-renders or stale closures for the typing loop.
- **Reveal Component**: This will be the standard wrapper for *all* future sections to ensure consistent entrance animations.
- **Typing Cursor**: Add a blinking cursor effect using CSS animation (e.g., `animate-pulse` or custom keyframes).
