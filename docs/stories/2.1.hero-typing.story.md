# Story 2.1: Build Hero Section with Typing Animation

## Status
Draft

## Story
**As a** Developer,
**I want** An engaging hero section with animated typing effect,
**so that** Visitors get a dynamic first impression of the portfolio

## Acceptance Criteria
1. Framer Motion installed and configured
2. `useTypingEffect` custom hook created
3. Reveal component for scroll animations (reusable)
4. Hero section with typing effect and CTA buttons
5. Full viewport height, centered content

## Tasks / Subtasks
- [ ] Task 1: Setup Animations (AC: 1, 3)
  - [ ] Install `framer-motion`
  - [ ] Create `src/components/common/Reveal.jsx`
  - [ ] Implement `whileInView` animation (fade up) with `once: true`
- [ ] Task 2: Create Typing Hook (AC: 2)
  - [ ] Create `src/hooks/useTypingEffect.js`
  - [ ] Implement robust typing/deleting logic with configurable speeds
  - [ ] Handle infinite loop of roles array
- [ ] Task 3: Build Hero Section (AC: 4, 5)
  - [ ] Create `src/components/sections/Hero.jsx`
  - [ ] Layout: `min-h-screen`, `flex`, `items-center`, `justify-center`
  - [ ] Typography: Large heading for name, dynamic subtext for roles
  - [ ] Integrate `useTypingEffect`
  - [ ] Add "View Work" (primary) and "Contact" (outline) buttons using `Button` component
  - [ ] Ensure responsive font sizes
- [ ] Task 4: Integration
  - [ ] Export `Hero` and add to `App.jsx` (inside Layout)
  - [ ] Verify buttons scroll to `#projects` and `#contact` placeholders

## Dev Notes
- **Framer Motion**: Standard library for React animations. [Source: docs/front-end-architecture.md#tech-stack]
- **Hook Logic**: Careful with `useEffect` dependencies to avoid infinite re-renders or stale closures for the typing loop.
- **Reveal Component**: This will be the standard wrapper for *all* future sections to ensure consistent entrance animations.
- **Typing Cursor**: Add a blinking cursor effect using CSS animation (e.g., `animate-pulse` or custom keyframes).

### Testing
- **Test File Location**: `src/hooks/__tests__/useTypingEffect.test.js`
- **Standards**: Verify hook cycles through words. Verify Hero renders without crashing.
