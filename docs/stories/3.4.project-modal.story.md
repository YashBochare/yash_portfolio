# Story 3.4: Build Project Detail Modal (Case Study)

## Status
Draft

## Story
**As a** Developer,
**I want** A modal showing full project case study,
**so that** Visitors can read detailed project information

## Acceptance Criteria
1. Reusable Modal component created
2. ProjectModal displays: Problem → Solution → Tech → Result
3. Close button and backdrop click to close
4. Scroll lock when modal open
5. Escape key closes modal
6. Smooth animations with Framer Motion

## Tasks / Subtasks
- [ ] Task 1: Create Generic Modal (AC: 1, 3, 4, 5, 6)
  - [ ] Create `src/components/common/Modal.jsx`
  - [ ] Use `createPortal` (optional, but recommended for modals) or fixed overlay
  - [ ] Framer Motion: `initial={{opacity: 0}} animate={{opacity: 1}} exit={{opacity: 0}}`
  - [ ] Scroll Lock: `useEffect` adding `overflow-hidden` to body
  - [ ] Escape Key: `useEffect` listener
  - [ ] Backdrop click handler
- [ ] Task 2: Create ProjectModal (AC: 2)
  - [ ] Create `src/components/common/ProjectModal.jsx` (or within Projects section if specific)
  - [ ] Consume `Modal`
  - [ ] Render project details: Large image, Title, structured Case Study sections
- [ ] Task 3: Integration
  - [ ] Add state `selectedProject` to `Projects.jsx`
  - [ ] Pass `setSelectedProject` to `ProjectCard`
  - [ ] Render `ProjectModal` when state is not null

## Dev Notes
- **Portal**: Using `ReactDOM.createPortal(content, document.body)` avoids z-index stacking context issues.
- **AnimatePresence**: Require wrapping the modal conditional in `AnimatePresence` in the *parent* component for exit animations to work.

### Testing
- **Test File Location**: `src/components/common/__tests__/Modal.test.jsx`
- **Standards**: Verify open/close behavior.
