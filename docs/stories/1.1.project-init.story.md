# Story 1.1: Setup Project Repository & Tech Stack

## Status
Approved

## Story
**As a** Developer,
**I want** A fully configured React + Vite + Tailwind project,
**so that** I can begin implementing components with proper tooling

## Acceptance Criteria
1. Vite React project initialized
2. Tailwind CSS configured with dark mode support
3. Project folder structure created
4. ESLint and Prettier configured
5. Git repository initialized with clean commit

## Tasks / Subtasks
- [ ] Task 1: Initialize Project (AC: 1)
  - [ ] Run `npm create vite@latest . -- --template react`
  - [ ] Install dependencies `npm install`
  - [ ] Clean up default Vite boilerplate (App.css, existing App.jsx content)
- [ ] Task 2: Configure Tailwind CSS (AC: 2)
  - [ ] Install Tailwind: `npm install -D tailwindcss postcss autoprefixer`
  - [ ] Init Tailwind: `npx tailwindcss init -p`
  - [ ] Configure `tailwind.config.js` with `darkMode: 'class'` and content paths
  - [ ] Add directives to `src/index.css`
  - [ ] Verify Tailwind works by adding a test class
- [ ] Task 3: Setup Project Structure (AC: 3)
  - [ ] Create directories: `src/components/common`, `src/components/layout`, `src/components/sections`, `src/context`, `src/data`, `src/hooks`, `src/styles`
  - [ ] Add `.gitkeep` if directories are empty
- [ ] Task 4: Configure Tools (AC: 4)
  - [ ] Install Prettier: `npm install -D prettier`
  - [ ] Create `.prettierrc` with project standards
  - [ ] Ensure ESLint is configured (Vite default is usually sufficient, verify `.eslintrc.cjs`)
  - [ ] Create `.vscode/settings.json` for workspace format-on-save (optional but good)
- [ ] Task 5: Git Initialization (AC: 5)
  - [ ] `git init`
  - [ ] Create `.gitignore` (add node_modules, .env.local, dist, .DS_Store)
  - [ ] Stage all files
  - [ ] Initial commit
